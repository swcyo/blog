## **扩展语法**

这些元素通过添加额外的功能扩展了基本语法。但是，并非所有 Markdown 应用程序都支持这些元素。

<table>
<thead>
<tr class="header">

<th><p>Markdown 语法</p></th>
</tr>
</thead>
<tbody>
<tr class="odd">

</tr>
<tr class="even">
<td><p><a href="https://markdown.com.cn/extended-syntax/fenced-code-blocks.html"><strong>代码块（Fenced Code Block）</strong></a></p></td>

</tr>
<tr class="odd">
